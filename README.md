## Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

#### 1. Clone repository

```Shell
git clone https://github.com/hugo-HDSF/M1PEX_DD_svelte.git
cd M1PEX_DD_svelte
nvm install
nvm use
npm run dev
```

#### 2. Open your browser and navigate to http://localhost:5173
